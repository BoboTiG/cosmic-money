MoneyBuster est un gestionnaire de dépenses partagées qui peut fonctionner
hors ligne ou se synchroniser avec IHateMoney et Nextcloud Payback.

En quoi est-il différent des autres gestionnaires de dépenses partagées ?

Vous pouvez garder votre projet local ou le synchroniser avec IHateMoney et Nextcloud Payback.
Cela veut dire que vous pouvez choisir où vont vos données et préserver votre vie privée.

# Fonctionnalités

* gérer les projets (ajouter/enlever/créer/supprimer/éditer)
* gérer les membres (ajouter/enlever/éditer)
* gérer les factures (ajouter/enlever/éditer)
* chercher des factures (par payeur, nom, montant, date)
* statistiques de projet
* plan de remboursement de projet
* partager les statistiques et plan de remboursement
* thème sombre
* interface utilisateur multi-langues (traduit sur [https://crowdin.com/project/moneybuster Crowdin])

# Dépendances

Si vous voulez héberger un projet sur IHateMoney :

* Une instance IHateMoney en fonctionnement

Si vous voulez héberger un projet sur Nextcloud Payback :

* Une instance Nextcloud en fonctionnement avec l'appli Payback installée

Si vous voulez pouvoir créer des projets distants depuis MoneyBuster : activez la création publique de projet sur votre instance IHateMoney ou Nextcloud Payback.

L'appli fonctionne sur Android>=4.1

# Alternatives

Tricount, Cospender, Splitwise etc... qui ne sont pas des FOSS
(Free or Open Source Software) et vous obligent à leur donner vos données privées.
